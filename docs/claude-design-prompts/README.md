## Quick start

1. Прочитайте `01-WORKFLOW.md` (5 минут)
2. Снимите подготовительные скриншоты (3 эталона + 4 для слайда 7)
3. Открыть `slide-09-business.md` первым (рекомендуется как стилевой якорь)
4. Cmd+A → Cmd+C → Cmd+V в новую сессию Claude Design
5. Прикрепить ассеты согласно списку в файле
6. Сгенерировать, отшлифовать, сохранить скрин финального результата как новый референс
7. Перейти к следующему слайду в новой сессии
